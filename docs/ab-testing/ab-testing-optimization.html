AB Testing is an Optimization Problem

Significance Testing is Costing You Revenue

TL;DR

Significance testing was designed for science — where false positives are costly and claims must hold up to scrutiny. It does that job well. But running a product experiment isn’t a scientific claim. It’s a business decision.

Three ways significance testing fails that decision:

  1. Significance testing optimizes for error control, not value. α controls false positives; β controls false negatives. Neither tells you whether acting on the result is worth it — an error around a negligible effect is irrelevant from a business perspective.
  2. Significance testing assumes a single fixed effect. The MDE — Minimum Detectable Effect — requires you to declare the effect size before the experiment. But treatment effects aren’t fixed. They come from a distribution — sometimes beneficial, sometimes harmful, usually marginal.
  3. Significance testing defaults to control. When results are inconclusive, significance testing says “keep control” — even when the data already points elsewhere.

Finding the Optimal Stopping Rule

Instead of α, β, and MDE, we fix:

  • Max Test N — the maximum number of users in the experiment
  • Post-Test N — the audience that sees the winner after the test ends

What we simulate:

  • Treatment effects drawn from a realistic distribution — not a single MDE

The question is simple: for a given experiment setup, what’s the optimal stopping rule? We use simulations to find out.

Optimal Stopping Algorithm

The same simulated data is evaluated under 50 different stopping rules — Pr(Best) thresholds from 0.50 to 0.99. For each:

  1. Burn-in: No stopping before day min_test_days = 7
  2. Daily check: If either condition’s Pr(Best) crosses the threshold, the experiment stops.
  3. Rollout: The leading condition is deployed to remaining test users and the full post-test audience
  4. Score: Record the long-run value produced

The threshold with the highest average score across all simulations is the optimal stopping rule.

    Conclusion

    Significance testing optimizes for error control, not revenue. It runs longer than necessary, defaults to control when the data already points elsewhere, and treats a threshold with no connection to long-run value as the decision criterion.

    The simulations show the cost. Compared to conventional significance testing (α = 0.05), Optimal Stopping ships the right variant more often, in fewer days, with more conversions over the full rollout window. The right stopping rule maximizes expected value, not statistical certainty.

    ⚡ 100,000 simulations × 20 days × 2 conditions — 2.4s

    To run your own optimization, swap in your experiment parameters — baseline conversion, test N, post-test N, effect distribution — and the pipeline handles the rest.

    The whole thing is fully vectorized. Every simulation, every day, every threshold is computed as a single set of array operations — no Python for loops anywhere. Under the hood, simulationengine uses Polars for columnar data transformations and NumPy for the numerical sweep.

    Four steps: simulate 100\,000 experiments, compute daily running stats, score each potential stopping decision, then sweep every Pr(Best) threshold to find the one that maximizes long-run value.

    import simulationengine as se
import numpy as np
import polars as pl

# ── Parameters ────────────────────────────────────────────────────────
CONTROL_MEAN       = 0.60      # baseline conversion rate
MAX_TEST_N         = 100_000   # total users in the experiment
SPLIT              = 0.50      # 50/50 control/treatment split
MIN_TEST_DAYS      = 7         # burn-in: no stopping before day 7
NUMBER_TEST_DAYS   = 20        # max experiment duration
POST_TEST_N        = 100_000   # post-test audience size
NUMBER_SIMULATIONS = 100_000
SEED               = 42
ITC_MEAN           = 100       # index-to-control mean (100 = no lift)
ITC_SE             = 1         # spread of treatment effects

# ── Step 1: Simulate experiments ──────────────────────────────────────
df = se.create_simulation_df(
    number_test_days=NUMBER_TEST_DAYS,
    number_simulations=NUMBER_SIMULATIONS,
    seed=SEED,
    condition_parameters=[{"mean": CONTROL_MEAN, "trials": int(MAX_TEST_N * SPLIT)}],
)
df = se.add_random_effect_condition(df, {
    "itc_mean": ITC_MEAN,
    "itc_se": ITC_SE,
    "trials": int(MAX_TEST_N * SPLIT),
})

# ── Step 2: Compute running stats ─────────────────────────────────────
df = se.add_cumulative_columns(df)
df = se.compute_cumulative_std_rates(df)
df = se.compute_delta_stats(df)
df = se.compute_test_stats(df)

# ── Step 3: Score each stopping decision ──────────────────────────────
df = se.calculate_remaining_test_potential(df)
df = se.calculate_launch_potential(df, post_test_n=POST_TEST_N)
df = se.calculate_total_potential(df)

# ── Step 4: Find the optimal Pr(Best) threshold ───────────────────────
p_thresholds = np.arange(0.5, 1.00, 0.01)

df_prepped = se.prepare_stopping_decision_columns(
    df.filter(pl.col("day") >= MIN_TEST_DAYS),
    metric_direction_improvement="increase",
).sort(["simulation", "day"])

results = se.evaluate_probability_thresholds(
    df_prepped,
    threshold_values=p_thresholds,
    metric_direction_improvement="increase",
    pre_sorted=True,
    select_best_at_end=True,
)

optimal_threshold = p_thresholds[results["mean_potential_value"].arg_max()]
print(f"Optimal Pr(Best) threshold: {optimal_threshold:.2f}")

    create_simulation_df generates a DataFrame of daily binomial draws for one or more fixed-rate conditions. Each row is one day of one simulation.

    import numpy as np
import polars as pl


def create_simulation_df(
    number_test_days,
    number_simulations,
    condition_parameters,
    seed=None,
):
    if seed is not None:
        np.random.seed(seed)

    # trials_per_day is the same for every day in the experiment
    n = np.array(
        [c["trials"] // number_test_days for c in condition_parameters]
    )[:, None]
    p = np.array([c["mean"] for c in condition_parameters])[:, None]

    rep = number_test_days * number_simulations
    n_mat = np.repeat(n, rep, axis=1)
    p_mat = np.repeat(p, rep, axis=1)

    # One binomial draw per (condition, day, simulation)
    samples = np.random.binomial(n=n_mat, p=p_mat)

    return pl.DataFrame(
        {
            "simulation": np.arange(rep) // number_test_days + 1,
            "day": np.arange(rep) % number_test_days + 1,
            **{
                f"condition_{i}_value": samples[i]
                for i in range(len(condition_parameters))
            },
            **{
                f"condition_{i}_trials": n_mat[i]
                for i in range(len(condition_parameters))
            },
            **{
                f"condition_{i}_true_mean": np.full(
                    rep, condition_parameters[i]["mean"]
                )
                for i in range(len(condition_parameters))
            },
        }
    )

    add_random_effect_condition appends a treatment arm where the true effect varies across simulations but is fixed within one. The effect is drawn from an ITC (Index-to-Control) distribution: ITC = 100 → no lift; ITC ~ N(100, 1) → effects centered at parity with modest spread.

    import numpy as np
import polars as pl


def add_random_effect_condition(df, random_effect_parameters, seed=None):
    if seed is not None:
        np.random.seed(seed)

    n_simulations = int(df["simulation"].max())
    n_days = int(df["day"].max())

    # One ITC draw per simulation — the unknown true effect for that experiment
    itc_samples = (
        np.random.normal(
            random_effect_parameters["itc_mean"],
            random_effect_parameters["itc_se"],
            n_simulations,
        )
        / 100
    )

    # treatment_mean = (itc / 100) x control_mean
    control_mean = float(df["condition_0_true_mean"][0])
    treatment_means = itc_samples * control_mean

    # Daily binomial draws; treatment_mean is constant across days within a simulation
    trials_per_day = random_effect_parameters["trials"] // n_days
    events = np.random.binomial(
        n=trials_per_day,
        p=treatment_means[:, None],
        size=(n_simulations, n_days),
    )

    # Reshape to tall format to match the existing DataFrame
    next_condition = (
        max(
            int(c.split("_")[1])
            for c in df.columns
            if c.startswith("condition_") and c.split("_")[1].isdigit()
        )
        + 1
    )
    return df.with_columns(
        [
            pl.Series(
                f"condition_{next_condition}_value",
                events.flatten(),
            ),
            pl.Series(
                f"condition_{next_condition}_trials",
                np.full(n_simulations * n_days, trials_per_day),
            ),
            pl.Series(
                f"condition_{next_condition}_true_mean",
                np.repeat(treatment_means, n_days),
            ),
        ]
    )

    add_cumulative_columns converts daily event counts into running totals grouped by simulation. Each condition gets a cumulative event count, trial count, and conversion rate. Cross-condition totals and remaining trials are derived in a single vectorized pass — no loops.

    import polars as pl


def add_cumulative_columns(df):
    value_cols  = [col for col in df.columns if col.endswith("_value")]
    trials_cols = [col for col in df.columns if col.endswith("_trials")]

    # Running totals per condition, grouped by simulation
    cumsum_exprs = [
        pl.col(col).cum_sum().over("simulation").alias(f"{col}_cumsum")
        for col in value_cols + trials_cols
    ]
    df = df.with_columns(cumsum_exprs)

    # Cross-condition totals
    value_cumsums  = [f"{c}_cumsum" for c in value_cols]
    trials_cumsums = [f"{c}_cumsum" for c in trials_cols]
    df = df.with_columns([
        pl.sum_horizontal(value_cumsums).alias("total_value_cumsum"),
        pl.sum_horizontal(trials_cumsums).alias("total_trials_cumsum"),
    ])

    # Trials remaining in the experiment window
    df = df.with_columns([
        (pl.col("total_trials_cumsum").max().over("simulation")
         - pl.col("total_trials_cumsum")).alias("total_remaining_test_trials")
    ])

    # Cumulative conversion rate per condition
    mean_exprs = [
        (pl.col(f"{c}_cumsum") / pl.col(f"{c.replace('_value','_trials')}_cumsum"))
        .alias(f"{c.replace('_value','')}_mean_cumsum")
        for c in value_cols
        if f"{c.replace('_value','_trials')}" in trials_cols
    ]
    return df.with_columns(mean_exprs)

    compute_cumulative_std_rates applies the binomial standard deviation formula sqrt(p * (1 - p)) to each condition's running conversion rate. Note: this is the std of the proportion, not the standard error — divide by sqrt(n) to get SE.

    import polars as pl


def compute_cumulative_std_rates(df):
    mean_cumsum_cols = [col for col in df.columns if col.endswith("_mean_cumsum")]

    std_exprs = [
        (pl.col(col) * (1 - pl.col(col)))
        .sqrt()
        .alias(col.replace("_mean_cumsum", "_std_cumsum"))
        for col in mean_cumsum_cols
    ]
    return df.with_columns(std_exprs)

    compute_delta_stats computes two things for each treatment condition vs control: the standard error (SE = std / sqrt(n)) and the delta statistics (delta_mean = mean_i - mean_0, delta_se = sqrt(SE_i² + SE_0²)). These are the direct inputs to z-scores and p-values — this function does not compute those itself.

    import polars as pl


def compute_delta_stats(df):
    mean_cols = [c for c in df.columns if c.endswith("_mean_cumsum")]

    # SE = std / sqrt(trials) for every condition
    se_exprs = [
        (pl.col(f"condition_{i}_std_cumsum") / pl.col(f"condition_{i}_trials_cumsum").sqrt())
        .alias(f"condition_{i}_se_cumsum")
        for i in [int(c.split("_")[1]) for c in mean_cols if c.startswith("condition_")]
        if f"condition_{i}_std_cumsum" in df.columns
    ]
    df = df.with_columns(se_exprs)

    # Delta mean and delta SE for each treatment vs condition_0 (control)
    treatment_nums = [
        int(c.split("_")[1]) for c in mean_cols
        if c.startswith("condition_") and int(c.split("_")[1]) > 0
    ]
    delta_exprs = []
    for i in treatment_nums:
        delta_exprs += [
            (pl.col(f"condition_{i}_mean_cumsum") - pl.col("condition_0_mean_cumsum"))
            .alias(f"condition_{i}_delta_mean"),
            (pl.col(f"condition_{i}_se_cumsum").pow(2) + pl.col("condition_0_se_cumsum").pow(2))
            .sqrt()
            .alias(f"condition_{i}_delta_se"),
        ]
    return df.with_columns(delta_exprs)

    compute_test_stats converts delta statistics into probabilities and expected values using the normal CDF. For each treatment: Pr(treatment > control) = Φ(delta_mean / delta_se). Expected gain and loss use truncated normal formulas. These are the columns the stopping rule reads — it stops when Pr(Best) crosses the threshold α.

    import numpy as np
import polars as pl
from scipy.stats import norm


def compute_test_stats(df):
    delta_mean_cols = [c for c in df.columns if c.endswith("_delta_mean")]
    condition_nums  = [int(c.split("_")[1]) for c in delta_mean_cols]

    pr_cols = []
    for i in condition_nums:
        z = df[f"condition_{i}_delta_mean"].to_numpy() / df[f"condition_{i}_delta_se"].to_numpy()
        pr = norm.cdf(z)
        df = df.with_columns([
            pl.lit(pr).alias(f"condition_{i}_pr_positive_diff"),
            pl.lit(1 - pr).alias(f"condition_{i}_pr_lowest_diff"),
        ])
        pr_cols.append(f"condition_{i}_pr_positive_diff")

        # Expected gain and loss (truncated normal)
        pdf_z = norm.pdf(z)
        df = df.with_columns([
            pl.lit(np.abs(pr   * (df[f"condition_{i}_delta_mean"].to_numpy() + df[f"condition_{i}_delta_se"].to_numpy() * pdf_z / np.maximum(pr,     1e-10)))).alias(f"condition_{i}_expected_gain"),
            pl.lit(np.abs((1-pr) * (-df[f"condition_{i}_delta_mean"].to_numpy() + df[f"condition_{i}_delta_se"].to_numpy() * pdf_z / np.maximum(1-pr, 1e-10)))).alias(f"condition_{i}_expected_loss"),
        ])

    # Control: Pr(best) = 1 - max(Pr(treatment > control))
    if pr_cols:
        df = df.with_columns([
            (1 - pl.max_horizontal(pr_cols)).alias("condition_0_pr_highest_diff"),
            pl.min_horizontal(pr_cols).alias("condition_0_pr_lowest_diff"),
        ])
    return df

    For each condition, draws from Binomial(total_remaining_test_trials, true_mean) — one random realization of the value you'd capture by stopping now and deploying that condition to remaining test users.

    import numpy as np
import polars as pl


def calculate_remaining_test_potential(df, seed=None):
    if not isinstance(df, pl.DataFrame):
        raise TypeError(f"df must be a polars DataFrame, got {type(df)}")
    if "total_remaining_test_trials" not in df.columns:
        raise ValueError("df must contain 'total_remaining_test_trials' column")
    if seed is not None:
        np.random.seed(seed)

    true_mean_cols = [col for col in df.columns if col.endswith("_true_mean")]
    if not true_mean_cols:
        raise ValueError("df must contain at least one column ending with '_true_mean'")

    potential_exprs = []
    for mean_col in true_mean_cols:
        condition_prefix = mean_col.replace("_true_mean", "")
        n = df["total_remaining_test_trials"].to_numpy().astype(int)
        p = np.clip(df[mean_col].to_numpy(), 0.0, 1.0)
        draws = np.random.binomial(n=n, p=p)
        potential_exprs.append(
            pl.lit(draws).alias(f"{condition_prefix}_test_potential_value")
        )

    return df.with_columns(potential_exprs)

    Draws from Binomial(post_test_n, true_mean) for each condition — one random realization of the value you'd capture by launching that condition to the post-test population. This is where stopping decisions have the most leverage.

def calculate_launch_potential(df, post_test_n, seed=None):
    if not isinstance(df, pl.DataFrame):
        raise TypeError(f"df must be a polars DataFrame, got {type(df)}")
    if not isinstance(post_test_n, (int, np.integer)):
        raise TypeError(f"post_test_n must be an integer, got {type(post_test_n)}")
    if post_test_n < 0:
        raise ValueError(f"post_test_n must be non-negative, got {post_test_n}")
    if seed is not None:
        np.random.seed(seed)

    true_mean_cols = [col for col in df.columns if col.endswith("_true_mean")]
    if not true_mean_cols:
        raise ValueError("df must contain at least one column ending with '_true_mean'")

    potential_exprs = []
    for mean_col in true_mean_cols:
        condition_prefix = mean_col.replace("_true_mean", "")
        p = np.clip(df[mean_col].to_numpy(), 0.0, 1.0)
        draws = np.random.binomial(n=post_test_n, p=p)
        potential_exprs.append(
            pl.lit(draws).alias(f"{condition_prefix}_launch_potential_value")
        )

    return df.with_columns(potential_exprs)

    total_potential = total_value_cumsum + test_potential + launch_potential. The cumulative observed value is shared across all conditions; only the forward-looking components differ. The condition with the highest total potential is the one to deploy.

def calculate_total_potential(df):
    if not isinstance(df, pl.DataFrame):
        raise TypeError(f"df must be a polars DataFrame, got {type(df)}")
    if "total_value_cumsum" not in df.columns:
        raise ValueError("df must contain 'total_value_cumsum' column")

    test_potential_cols = [
        col for col in df.columns if col.endswith("_test_potential_value")
    ]
    if not test_potential_cols:
        raise ValueError("df must contain at least one column ending with '_test_potential_value'")

    total_exprs = []
    for test_col in test_potential_cols:
        condition_prefix = test_col.replace("_test_potential_value", "")
        launch_col = f"{condition_prefix}_launch_potential_value"
        if launch_col not in df.columns:
            raise ValueError(f"Missing required column: {launch_col}")
        total_exprs.append(
            (pl.col("total_value_cumsum") + pl.col(test_col) + pl.col(launch_col))
            .alias(f"{condition_prefix}_total_potential_value")
        )

    return df.with_columns(total_exprs)

    The core of the optimization loop. Rather than iterating over thresholds in Python, it stacks all Pr(Best) columns into a 3D NumPy array (n_sims × n_days × n_conditions), finds the first crossing day per simulation per threshold in a single broadcast, then aggregates value, trial count, and decision accuracy across all thresholds simultaneously.

    import numpy as np
import polars as pl


def evaluate_probability_thresholds(
    df_prepped,
    threshold_values,
    metric_direction_improvement="increase",
    pre_sorted=False,
    select_best_at_end=False,
):
    if not isinstance(df_prepped, pl.DataFrame):
        raise TypeError(f"df_prepped must be a polars DataFrame, got {type(df_prepped)}")
    if metric_direction_improvement not in ["increase", "decrease"]:
        raise ValueError(f"metric_direction_improvement must be 'increase' or 'decrease'")
    if not pre_sorted:
        df_prepped = df_prepped.sort(["simulation", "day"])

    n_sims = df_prepped["simulation"].n_unique()
    n_days = df_prepped.height // n_sims

    def _col2d(name):
        return df_prepped[name].to_numpy().reshape(n_sims, n_days)

    # Select probability columns based on metric direction
    if metric_direction_improvement == "increase":
        treatment_prob_cols = [
            c for c in df_prepped.columns
            if c.endswith("_pr_positive_diff") and not c.startswith("condition_0_")
        ]
        control_prob_col = "condition_0_pr_highest_diff"
    else:
        treatment_prob_cols = [
            c for c in df_prepped.columns
            if c.endswith("_pr_lowest_diff") and not c.startswith("condition_0_")
        ]
        control_prob_col = "condition_0_pr_lowest_diff"

    all_prob_cols = [control_prob_col] + treatment_prob_cols
    condition_nums = np.array([int(c.split("_")[1]) for c in all_prob_cols])

    # Stack: (n_sims, n_days, n_conditions)
    prob_stack = np.stack([_col2d(c) for c in all_prob_cols], axis=2)
    max_prob   = prob_stack.max(axis=2)            # (n_sims, n_days)
    best_cond  = condition_nums[prob_stack.argmax(axis=2)]

    # Cumulative max — first day cummax >= threshold is the stopping day
    cummax = np.maximum.accumulate(max_prob, axis=1)
    thresholds_arr = np.array(sorted(threshold_values))

    # First-crossing index, clamped to last day for never-crossed sims
    raw_stop   = (cummax[:, :, np.newaxis] < thresholds_arr).sum(axis=1)  # (n_sims, n_thresholds)
    any_crossed = raw_stop < n_days
    stop_idx   = np.minimum(raw_stop, n_days - 1)

    row = np.arange(n_sims)[:, np.newaxis]
    best_cond_at_stop = best_cond[row, stop_idx]
    winning_cond = np.where(any_crossed | select_best_at_end, best_cond_at_stop, 0)

    # Winning condition's total potential value at stopping day
    pot_cols = sorted(
        [c for c in df_prepped.columns
         if c.endswith("_total_potential_value") and c.startswith("condition_") and c.split("_")[1].isdigit()],
        key=lambda c: int(c.split("_")[1]),
    )
    pot_stack      = np.stack([_col2d(c) for c in pot_cols], axis=2)
    win_pot_at_stop = pot_stack[row, stop_idx, winning_cond]

    total_at_stop    = _col2d("total_trials_cumsum")[row, stop_idx]
    true_cond_at_stop = _col2d("true_best_condition")[row, stop_idx]

    return pl.DataFrame({
        "threshold":           thresholds_arr,
        "mean_potential_value": win_pot_at_stop.mean(axis=0),
        "mean_trials":          total_at_stop.mean(axis=0),
        "median_trials":        np.median(total_at_stop, axis=0),
        "p80_trials":           np.percentile(total_at_stop, 80, axis=0),
        "match_rate":           (winning_cond == true_cond_at_stop).mean(axis=0),
    })

# TL;DR

Significance testing was designed for science — where false positives are costly and claims must hold up to scrutiny. It does that job well. But running a product experiment isn't a scientific claim. It's a business decision.

Three ways significance testing fails that decision:

1. **Significance testing optimizes for error control, not value.** α controls false positives; β controls false negatives. Neither tells you whether acting on the result is worth it — an error around a negligible effect is irrelevant from a business perspective.
2. **Significance testing assumes a single fixed effect.** The MDE — Minimum Detectable Effect — requires you to declare the effect size before the experiment. But treatment effects aren't fixed. They come from a distribution — sometimes beneficial, sometimes harmful, usually marginal.
3. **Significance testing defaults to control.** When results are inconclusive, significance testing says "keep control" — even when the data already points elsewhere.



# Finding the Optimal Stopping Rule

Instead of α, β, and MDE, we fix:

- **`Max Test N`** — the maximum number of users in the experiment
- **`Post-Test N`** — the audience that sees the winner after the test ends

What we simulate:

- Treatment effects drawn from a realistic distribution — not a single MDE

The question is simple: for a given experiment setup, what's the optimal stopping rule? We use simulations to find out.

## Optimal Stopping Algorithm

The same simulated data is evaluated under 50 different stopping rules — Pr(Best) thresholds from 0.50 to 0.99. For each:

1. **Burn-in:** No stopping before day `min_test_days = 7`
2. **Daily check:** If either condition's Pr(Best) crosses the threshold, the experiment stops.
3. **Rollout:** The leading condition is deployed to remaining test users and the full post-test audience
4. **Score:** Record the long-run value produced

The threshold with the highest average score across all simulations is the optimal stopping rule.

# Conclusion

Significance testing optimizes for error control, not revenue. It runs longer than necessary, defaults to control when the data already points elsewhere, and treats a threshold with no connection to long-run value as the decision criterion.

The simulations show the cost. Compared to conventional significance testing (α = 0.05), Optimal Stopping ships the right variant more often, in fewer days, with more conversions over the full rollout window. **The right stopping rule maximizes expected value, not statistical certainty.**

from scipy.stats import norm


def compute_test_stats(df):
    delta_mean_cols = [c for c in df.columns if c.endswith("_delta_mean")]
    condition_nums  = [int(c.split("_")[1]) for c in delta_mean_cols]

    pr_cols = []
    for i in condition_nums:
        z = df[f"condition_{i}_delta_mean"].to_numpy() / df[f"condition_{i}_delta_se"].to_numpy()
        pr = norm.cdf(z)
        df = df.with_columns([
            pl.lit(pr).alias(f"condition_{i}_pr_positive_diff"),
            pl.lit(1 - pr).alias(f"condition_{i}_pr_lowest_diff"),
        ])
        pr_cols.append(f"condition_{i}_pr_positive_diff")

        # Expected gain and loss (truncated normal)
        pdf_z = norm.pdf(z)
        df = df.with_columns([
            pl.lit(np.abs(pr   * (df[f"condition_{i}_delta_mean"].to_numpy() + df[f"condition_{i}_delta_se"].to_numpy() * pdf_z / np.maximum(pr,     1e-10)))).alias(f"condition_{i}_expected_gain"),

        ])

    if pr_cols:

    <pre style="background:#f8f8f8; border-radius:4px; padding:1rem; font-size:0.8rem; overflow-x:auto;"><code>import numpy as np
import polars as pl


def evaluate_probability_thresholds(
    df_prepped,
    threshold_values,
    metric_direction_improvement="increase",
    pre_sorted=False,
    select_best_at_end=False,
):
    if not isinstance(df_prepped, pl.DataFrame):
        raise TypeError(f"df_prepped must be a polars DataFrame, got {type(df_prepped)}")
    if metric_direction_improvement not in ["increase", "decrease"]:
        raise ValueError(f"metric_direction_improvement must be 'increase' or 'decrease'")
    if not pre_sorted:
        df_prepped = df_prepped.sort(["simulation", "day"])

    n_sims = df_prepped["simulation"].n_unique()
    n_days = df_prepped.height // n_sims

    def _col2d(name):
        return df_prepped[name].to_numpy().reshape(n_sims, n_days)

    # Select probability columns based on metric direction
    if metric_direction_improvement == "increase":
        treatment_prob_cols = [
            c for c in df_prepped.columns
            if c.endswith("_pr_positive_diff") and not c.startswith("condition_0_")
        ]
        control_prob_col = "condition_0_pr_highest_diff"
    else:
        treatment_prob_cols = [
            c for c in df_prepped.columns
            if c.endswith("_pr_lowest_diff") and not c.startswith("condition_0_")
        ]
        control_prob_col = "condition_0_pr_lowest_diff"

    all_prob_cols = [control_prob_col] + treatment_prob_cols
    condition_nums = np.array([int(c.split("_")[1]) for c in all_prob_cols])

    # Stack: (n_sims, n_days, n_conditions)
    prob_stack = np.stack([_col2d(c) for c in all_prob_cols], axis=2)
    max_prob   = prob_stack.max(axis=2)            # (n_sims, n_days)
    best_cond  = condition_nums[prob_stack.argmax(axis=2)]

    # Cumulative max — first day cummax >= threshold is the stopping day
    cummax = np.maximum.accumulate(max_prob, axis=1)
    thresholds_arr = np.array(sorted(threshold_values))

    # First-crossing index, clamped to last day for never-crossed sims
    raw_stop   = (cummax[:, :, np.newaxis] &lt; thresholds_arr).sum(axis=1)  # (n_sims, n_thresholds)
    any_crossed = raw_stop &lt; n_days
    stop_idx   = np.minimum(raw_stop, n_days - 1)

    row = np.arange(n_sims)[:, np.newaxis]
    best_cond_at_stop = best_cond[row, stop_idx]
    winning_cond = np.where(any_crossed | select_best_at_end, best_cond_at_stop, 0)

    # Winning condition's total potential value at stopping day
    pot_cols = sorted(
        [c for c in df_prepped.columns
         if c.endswith("_total_potential_value") and c.startswith("condition_") and c.split("_")[1].isdigit()],
        key=lambda c: int(c.split("_")[1]),
    )
    pot_stack      = np.stack([_col2d(c) for c in pot_cols], axis=2)
    win_pot_at_stop = pot_stack[row, stop_idx, winning_cond]

    total_at_stop    = _col2d("total_trials_cumsum")[row, stop_idx]
    true_cond_at_stop = _col2d("true_best_condition")[row, stop_idx]

    return pl.DataFrame({
        "threshold":           thresholds_arr,
        "mean_potential_value": win_pot_at_stop.mean(axis=0),
        "mean_trials":          total_at_stop.mean(axis=0),
        "median_trials":        np.median(total_at_stop, axis=0),
        "p80_trials":           np.percentile(total_at_stop, 80, axis=0),
        "match_rate":           (winning_cond == true_cond_at_stop).mean(axis=0),
    })</code></pre>
